Blender Archive

Welcome to the Blender internet archive! In this portal you can access legacy versions of the Blender online infrastructure. The data is all static for historical accuracy, and security.

blender.org

The official Blender website. Originally running on Typo3.

Archived: 2012

wiki.blender.org

The legacy Blender Wiki, using custom Naiad skin. This website used to host the user manual as well.

Archived: 2015

developer.blender.org

Developers portal, running on Phabricator, for repo management, patch review and bug triaging.

Archived: 2023
